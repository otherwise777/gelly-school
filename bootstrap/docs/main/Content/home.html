Tutorials

Tutorial0
Introduction and all you need to get started with Gelly.

Tutorial1
Load public data and create a graph and run a library method: It deals with graph creation methods in Gelly. A publicly available graph dataset is used to read a file of edges, create, and initialize a graph in Gelly.

Tutorial2
Know your graph: Use the Gelly methods to get some metrics and statistics out of a graph e.g., number of vertices, edges and node degrees.

Tutorial3
Simple transformations: Learn how to transform a graph into another graph using Gelly's transformation methods like filtering and mapping.

Tutorial4
Advanced Transformations: Some more transformations e.g., perform a join, union

Tutorial5
Neighborhood methods: It describe Gelly's neighborhood methods and how to use them for computing neighborhood aggregations. As an example, a naive version of clustering coefficient computation is used.

Tutorial6
Implement a graph validator: Learn to check the validity of a graph e.g., if all edge weights are positive.

Tutorial7
Vertex-centric iterations: It explain the vertex-centric model and how this is implemented in Gelly. As an example,a naive PageRank algorithm is implemented.

Tutorial8
Gather-sum-apply iterations: It explains the GSA model and how it is implemented in Gelly. PageRank algorithm is also used here and a comparison between the two iterative models is given.
