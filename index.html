Welcome to Gellyschool!

Gellyschool is a series of tutorials on Apache Flink's Graph API, Gelly.
Here you can find step-by-step tutorials on how to create and analyze your graphs with Gelly.
For a complete reference to Gelly methods and library, please visit the Gelly guide. Enjoy your time at Gellyschool!

Tutorials

Tutorial#0
Graphs 101: What is a Graph and why should you care?

Tutorial#1
Create a Graph and run a Gelly library method: Learn how to create Graphs from text files and how to use the available a Gelly library methods.

Tutorial#2
Know your Graph: Get metrics and statistics out of your Graph. How many vertices does it have? What are their degrees?

Tutorial#3
Simple transformations: Transform your Graph into another Graph with different properties. Learn how to use filtering and mapping methods.

Tutorial#4
Advanced transformations: Use more advanced transformations, like join, union and difference.

Tutorial#5
Neighborhood methods: Learn about Gelly's neighborhood methods and how you can use them to compute neighborhood aggregations. Implement a Local Clustering Coefficient algorithm.

Tutorial#6
Graph validation: Is your Graph valid? Implement a validator to find out.

Tutorial#7
Vertex-centric iterations: Learn about the vertex-centric graph processing model and implement your first vertex-centric iteration in Gelly.

Tutorial#8
Gather-sum-apply iterations: Learn about the gather-sum-apply graph processing model and implement you first GSA iteration in Gelly.
