Implement a graph validator

Don't trust your Graph

Now that we know how to define a Graph in Gelly and to perform a good amount of transformations on it, we might want to check the Graph validity. Gelly provides a simple utility for performing validation checks on input graphs. Depending on the application context, a graph may or may not be valid according to certain criteria e.g., a user might need to validate whether their graph contains duplicate edges.
Validity checks can be performed in two ways:

  • Use InvalidVertexIdsValidator() which is Gellyâ€™s pre-defined validator. It checks that the edge set contains valid vertex IDs, i.e. that all edge IDs also exist in the vertex IDs set.

    •     Boolean validGraph;
    validGraph = graph.validate(new InvalidVertexIdsValidator<Long,Double,Double>());

  • Define a custom GraphValidator and implement its validate() method.

    • Below is an example of a custom validator which shows how the validate method can be defined. In this example, the method checks whether all the edges have been assigned a value or not. In case there exists an edge with null value, the CheckEdgeIds() returns a non null set of edges, so the validate method returns false. You can implement any desired check by just modifying the validate method.


        validGraph= graph.validate(new InvalidEdgeValidator());


        @SuppressWarnings("serial")
        static final class InvalidEdgeValidator extends GraphValidator<Long,Double,Double> {

            @Override
	    //return valid only if there is no edge with null value
            public boolean validate(Graph<Long, Double, Double> graph)
                    throws Exception {      
                DataSet<Edge<Long,Double>> edgeIds = graph.getEdges()
                    .flatMap(new CheckEdgeIds<Long,Double>());
                return edgeIds.count()==0;
            }
            //return edges with null values
            private static final class CheckEdgeIds<Long,Double> implements FlatMapFunction<Edge<Long,Double>,
    		Edge<Long,Double>> {
                public void flatMap(Edge<Long,Double> edge, Collector<Edge<Long,Double>> out) {
                    if(edge.getValue() == null){ 
                    out.collect(new Edge<Long,Double>(edge.f0,edge.f1,edge.f2));
                    }

                }
            }
